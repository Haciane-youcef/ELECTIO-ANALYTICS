# Electio-Analytics

Projet MSPR EPSI – API de prédiction électorale basée sur le Machine Learning.

## Technologies utilisées

- Python
- FastAPI
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Fonctionnalités

- Prédiction du gagnant électoral
- Classement des candidats
- API REST avec FastAPI
- Frontend interactif avec Streamlit

## Lancer l'API

```bash
python -m uvicorn api:app --reload
```

## Lancer le frontend

```bash
python -m streamlit run frontend.py
```
